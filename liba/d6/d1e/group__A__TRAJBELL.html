generate for bell-shaped velocity trajectory

If \(q_0>q_1\), then

\begin{aligned}q_0=-q_0\\q_1=-q_1\\v_0=-v_0\\v_1=-v_1\end{aligned}

  1. If \((v_m-v_0)j_m<a_m^2\), the maximum acceleration is not reached, then

    \begin{aligned}T_{aj}&=\sqrt{\frac{v_m-v_0}{j_m}}\\T_a&=2T_{aj}\\a_m&=+j_mT_{aj}\end{aligned}

    else

    \begin{aligned}T_{aj}&=\frac{a_m}{j_m}\\T_a&=T_{aj}+\frac{v_m-v_0}{a_m}\\a_m&=+a_m\end{aligned}

  2. If \((v_m-v_1)j_m<a_m^2\), the maximum deceleration is not reached, then

    \begin{aligned}T_{dj}&=\sqrt{\frac{v_m-v_1}{j_m}}\\T_d&=2T_{dj}\\d_m&=-j_mT_{dj}\end{aligned}

    else

    \begin{aligned}T_{dj}&=\frac{a_m}{j_m}\\T_d&=T_{dj}+\frac{v_m-v_1}{a_m}\\d_m&=-a_m\end{aligned}

  3. Calculate the constant velocity phase

    \[T_v=\frac{q_1-q_0}{v_m} -\frac{T_a}{2}\left(1+\frac{v_0}{v_m}\right)-\frac{T_d}{2}\left(1+\frac{v_1}{v_m}\right)\]

    If \(T_v>0\), then there is a constant velocity phase and the phase calculation ends.
  4. If \(T_v<=0\), then \(T_v=0\),

    \begin{aligned}T_{aj}&=T_{dj}=T_{j}=\frac{a_m}{j_m}\\ T_a&=\cfrac{\frac{a_m^2}{j_m}+\sqrt\Delta-2v_0}{2a_m}\\T_d&=\cfrac{\frac{a_m^2}{j_m}+\sqrt\Delta-2v_1}{2a_m}\\ \Delta&=\frac{a_m^4}{j_m^2}+2(v_0^2+v_1^2)+a_m\left(4(q_1-q_0)-2(v_0+v_1)\frac{a_m}{ j_m}\right)\end{aligned}


    a. If \(T_a<0\), there is only deceleration phase, then \(T_a=0\), \(T_{aj}=0\),

    \begin{aligned}T_d&=2\cfrac{q_1-q_0}{v_1+v_0}\\ T_{dj}&=\frac{j_m(q_1-q_0)-\sqrt{j_m(j_m(q_1-q_0)^2+(v_1-v_0)(v_1+v_0)^2)}}{j_m(v_1+v_0 )}\end{aligned}

    b. If \(T_a<0\), there is only acceleration phase, then \(T_d=0\), \(T_{dj}=0\),

    \begin{aligned}T_a&=2\cfrac{q_1-q_0}{v_1+v_0}\\ T_{aj}&=\frac{j_m(q_1-q_0)-\sqrt{j_m(j_m(q_1-q_0)^2+(v_0-v_1)(v_1+v_0)^2)}}{j_m(v_1+v_0 )}\end{aligned}

    c. If \(T_a\ge2T_j\), \(T_d\ge2T_j\), then

    \begin{aligned}a_m&=+j_mT_{aj}\\d_m&=-j_mT_{dj}\\v_m&=v_0+(T_a-T_{aj})a_m=v_1-(T_d-T_{dj})d_m\end{aligned}

    d. If none of the above conditions are met, let \(a_m=\alpha a_m, 0<\alpha<1\), and then repeat step 4.
  5. Finally, use formulas to calculate position, velocity and acceleration.
