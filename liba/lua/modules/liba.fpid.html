constructor for fuzzy PID controller

Parameters:

  • num int maximum number triggered by the rule
  • dt number sampling time unit(s)
  • mmp table points to membership function parameter table, an array terminated by 0
  • mkp table points to Kp's rule base table, the rule base must be square
  • mki table points to Ki's rule base table, the rule base must be square
  • mkd table points to Kd's rule base table, the rule base must be square
  • imin number minimum input
  • imax number maximum input
  • omin number minimum output
  • omax number maximum output
  • sum number maximum intergral output (optional)
