generate for trapezoidal velocity profile trajectory

Assuming that there is no constant velocity phase, but only acceleration and deceleration phases, the maximum velocity in the motion is \(v\), then we have

\[q=q_a+q_c=\frac{v^2-v_s^2}{2a}+\frac{v_e^2-v^2}{2d}\]

Solving for the maximum velocity is

\[|v|=\sqrt{\frac{av_e^2-dv_s^2-2qad}{a-d}}\]

  1. If \(|v|>|v_m|\), then there is a constant velocity phase.

    \[\begin{cases}t_a=\cfrac{v_m-v_s}{a}\\t_c=\cfrac{q-q_a-q_d}{v_c}\\t_d=\cfrac{v_e-v_m}{d}\end{cases}\]

  2. Otherwise there is no constant velocity phase.
    a. If \(|v_s|<|v|\le|v_e|\), there is only an acceleration phase.

    \[|v_e|=\sqrt{v_s^2+2qa}\]

    \[\begin{cases}t_a=\cfrac{v_e-v_s}{a}\\t_c=0\\t_d=0\end{cases}\]

    b. If \(|v_s|\ge|v|>|v_e|\), there is only a deceleration phase.

    \[|v_e|=\sqrt{v_s^2+2qd}\]

    \[\begin{cases}t_a=0\\t_c=0\\t_d=\cfrac{v_e-v_s}{d}\end{cases}\]

    c. If \(|v|>|v_s|\), \(|v|>|v_e|\), then there are acceleration and deceleration phases.

    \[\begin{cases}t_a=\cfrac{v-v_s}{a}\\t_c=0\\t_d=\cfrac{v_e-v}{d}\end{cases}\]

  3. Finally, the position and velocity are calculated using the formula.
